# fair-fly

FAIR FLY API

## Features
Allows to find out which flying sites from the DHV's site database are connected to FAIR FLY and what their status is.

## URL
api.fair-fly.com

## Prerequisitions
API Key, can be request by email to info@fair-fly.com.
Please provide the following information:
1. First name
2. Last name
3. Email address
4. Purpose of use

Emails containing incomplete information will be ignored.
